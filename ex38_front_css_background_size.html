background-size는 직사각형 테스트 권장

background-size: auto
auto는 원본 크기
background-size: 200px
200px은 가로 크기 200px, 세로는 원본 비율 유지
background-size: 50%
50%는 가로 크기 50%, 세로는 원본 비율 유지
background-size: 100% 100%
100% 100%는 가로 세로 모두 100%로 늘림
background-size: contain
contain은 요소 안에 이미지가 모두 보이도록 크기 조절
background-size: cover
cover는 요소를 완전히 덮도록 크기 조절, 이미지의 일부가 잘릴 수 있음
