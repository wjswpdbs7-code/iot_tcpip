상속은 같은 특징을 가지는 객체들을 그룹화하여 부모 클래스(슈퍼 클래스)로 정의하고, 
자식 클래스(서브 클래스)가 부모 클래스를 상속받아 공통된 특징을 공유하는 객체 지향 프로그래밍의 개념입니다.

단지 코드를 짧게 만들기 위해서가 아니라, 기초적인 특징을 부모 클래스에 정의하고, 
자식 클래스에서 추가적인 특징이나 기능을 확장할 수 있도록 하는 것이 상속의 주요 목적입니다. 

예시 

class 도형:
    변의 개수
    각의 개수

class 삼각형(도형):
    변의 개수 = 3
    각의 개수 = 3
    

class 사각형(도형):
    변의 개수 = 4
    각의 개수 = 4
